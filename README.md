# Performance Optimization of Python-Based AI Calculations Using Mojo

## Project Overview

Mojo is still a new and less popular programming language compared to Python.  
This project demonstrates how Mojo can be applied to a Python-based AI calculation project.

The project first runs a slow repeated AI-style calculation in Python.  
Then, the same performance-critical calculation is rewritten in Mojo.

## Main Objective

To compare the execution time between:

- Original Python calculation
- Optimized Mojo calculation

## Expected Output

```text
Original Python calculation completed in: 2.8 seconds
Optimized Mojo calculation completed in: 0.7 seconds
Performance improvement: 4x faster
